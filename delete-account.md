# アカウント削除リクエスト

**アプリ名 / App**: Seisany
**運営者 / Developer**: nujimasu

---

## 削除リクエストの方法（日本語）

Seisany アカウントおよび関連データの削除をご希望の場合、以下のいずれかの方法でリクエストしてください。

### 方法 1: アプリ内から削除（推奨）

1. Seisany アプリを起動
2. 設定（⚙ Settings）画面を開く
3. 「アカウント削除」を選択（※実装状況により設定画面のお問い合わせから連絡ください）

### 方法 2: メールでリクエスト

下記アドレスにアプリ内表示名と Apple ID / Google アカウントのいずれを使用しているかを記載のうえ送信してください。

**メール送信先**: [GitHub Issue](https://github.com/nujimasu/seisany-legal/issues/new) でも受け付けています

---

## 削除されるデータ

リクエスト受領後、**14日以内** に以下のデータを完全に削除します：

- 認証情報（Apple ID / Google アカウントID）
- 表示名
- 作成・参加したシェアグループのメンバー情報
- 立替・精算データ（ユーザーに紐づくもの）
- プッシュ通知トークン

## 保持されるデータ（法令遵守目的）

以下のデータは集計済み・匿名化された統計データとして最大 **90日間** 保持される場合があります：

- アプリの利用ログ（個人を特定しない形式）
- クラッシュレポート（端末情報のみ）

## ワンショットモードのデータ

ワンショットモードのデータはお使いの端末内にのみ保存されています。アプリをアンインストールするか、設定画面から「データのリセット」を実行することで削除できます。

# Account Deletion Request (English)

**App**: Seisany
**Developer**: nujimasu

## How to Request Deletion

### Option 1: In-App Deletion (Recommended)

1. Launch the Seisany app
2. Open Settings (⚙ Settings)
3. Tap "Delete Account" (or contact us via the in-app inquiry option)

### Option 2: Email or GitHub Issue

Please submit your request via [GitHub Issue](https://github.com/nujimasu/seisany-legal/issues/new), including your in-app display name and which sign-in provider you used (Apple / Google).

---

## Data That Will Be Deleted

We will permanently delete the following within **14 days** of receiving your request:

- Authentication credentials (Apple ID / Google account ID)
- Display name
- Member information in shared groups you created or joined
- Expense and settlement data linked to your account
- Push notification tokens

## Data Retained (for Legal Compliance)

The following may be retained as aggregated, anonymized statistics for up to **90 days**:

- App usage logs (in non-identifiable form)
- Crash reports (device info only)

## One-Shot Mode Data

One-Shot Mode data is stored only on your device. To delete it, simply uninstall the app or use the "Reset Data" option in Settings.
